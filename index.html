Flexbox Row

Use these three properties to create a flexbox row layout.

    .row { 
        display: flex;
         flex direction: colum;
      }

FlexBox Column

Use these three properties to create a flexbox column layout.

    .column { 
        display: flex;
         flex direction: colum;
      }

CSS Grid Layout

Build a 12 colum layout using CSS Grid.

    .grid { 
        display: grid;
         width: 100%
     grid-template-columns:
    repeat(12. 1fr)     
}
